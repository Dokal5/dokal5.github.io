Guided Reading Path

The sequence map above shows the two core versions side by side. What follows is a synthesized teaching structure: not a pure EVAC 1.0 shell and not a literal EVAC 2.0 protocol, but a clearer course architecture built from both.

The goal of this section is to turn the two frameworks into a teachable reading order. Each part below carries a light mapping note so readers can see which ideas come mainly from EVAC 1.0, which come mainly from EVAC 2.0, and where the synthesis happens.
