Pricing Cases

Pricing Cases is a live case library for counterintuitive pricing logic. It exists to support EVAC teaching, sharpen competitive pricing diagnosis, and turn real-world price structures into reusable strategic principles.

What This Library Is For

This is not a gallery of company price lists. Each case must explain how value is actually captured, where the logic is vulnerable, and what a strategist can learn from the structure itself.

Research for EVAC teaching Cases are mapped back to EVAC concepts so the course can reference current market evidence.
Competitive gap finding Each case must identify a flaw, threshold, exposure point, or design opening rather than stopping at description.
Consulting-grade pattern capture The library is structured so cases can be reused in client analysis, workshop design, and future cross-industry comparisons.
